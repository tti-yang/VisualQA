# Adding a page

No coding needed.

**Issues → New issue → "Submit a page."** Pick the modality and where in the
river, write the page, drag in any images. Done — a maintainer publishes it.

Prefer to write a file directly? Copy [`_artifacts/example-template.md`](_artifacts/example-template.md),
set the modality / placement at the top, write, and upload images with
**Add file → Upload files**. Edit it again anytime.
