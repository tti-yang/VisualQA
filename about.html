About VisualQA

Suggest a tool GitHub

Great tools shouldn't stay hidden.

The open science community builds incredible tools across every imaging modality — but they're scattered across GitHub orgs, lab websites, and word of mouth. Great work gets missed. This directory exists so it doesn't have to.

VisualQA is a shared overview of open-source software across imaging modalities. If everyone contributes what they know, it becomes a way for the whole community to stay updated and discover each other's work.

How health status works. A weekly GitHub Actions cron job checks each tool's repository.

  • Active — committed within 6 months
  • Quiet — 6 to 24 months
  • Inactive — 2+ years or archived
  • Not on GitHub

What gets included. Tools need to be relevant to their modality, open-source, and actively maintained. New or low-star tools are welcome as long as they solve a real problem and are being worked on.

How to contribute. Open an issue directly on GitHub, or submit a pull request. Each tool is a YAML entry: name, url, description, language, modality (top-level: MRI, EEG), optionally submodality (e.g. fMRI, diffusion) and github (owner/repo for health tracking), and added_by (with optional added_by_github handle) to credit whoever suggested it.
