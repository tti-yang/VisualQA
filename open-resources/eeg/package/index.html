EMEGS

ElectroMagnetic EncephaloGraphy Software

What it is: Open-source software for preprocessing, visualizing, and statistically analyzing psychophysiological EEG/MEG data. It is available as a MATLAB package and as a standalone Qt application.

Start here: First choose the MATLAB or standalone version, then check its documentation and example workflow. The standalone option is useful if you do not have MATLAB.

Autoreject

Automated rejection and repair for M/EEG epochs

What it is: A Python library that learns thresholds from M/EEG data to reject bad trials and repair bad sensors. It works directly with epoched MNE-Python data.

Start here: Read the short quickstart, then open the automatic repair example. Compare its cleaned epochs and rejection log with the raw data instead of treating the output as automatically correct.

MNE-Python

Open-source Python software for human neurophysiology

What it is: A broad Python package for importing, exploring, visualizing, preprocessing, and analyzing MEG, EEG, sEEG, ECoG, NIRS, and related data.

Start here: Follow the introductory tutorials in order: data structures, continuous data, events and epochs, preprocessing, then evoked responses. Use the example gallery when you need a task-specific workflow.

Data standard Organization & sharing

EEG-BIDS

Brain Imaging Data Structure specification for EEG

What it is: A community standard for consistently naming, organizing, and describing EEG datasets and their metadata. It is a specification, not an analysis package.

Start here: Begin with the BIDS Starter Kit, then use the EEG-specific page to check required files, accepted recording formats, electrode and coordinate metadata, events, and sidecar JSON fields.

Brainstorm

Open-source application for multimodal brain recordings

What it is: A free, open-source application for analyzing MEG, EEG, fNIRS, ECoG, depth-electrode, and electrophysiology data through a graphical interface, with scripting available for batch workflows.

Start here: Read the introduction, install the standalone version if you do not have MATLAB, and work through the EEG/ERP tutorials before building your own analysis pipeline.
